### 新增 (v0.5.0 Phase 5)
- 🎨 **UI/UX 优化**
  - LoadingSpinner 组件（spinner, dots, pulse 三种样式）
  - ErrorMessage 组件（error, warning, info 三种类型）
  - Toast 通知系统（成功/错误/警告/信息提示）
  - ConfirmDialog 确认对话框（danger, warning, info 三种样式）
  - 用户友好的错误提示和反馈

- ⌨️ **键盘快捷键增强**
  - `Ctrl/Cmd + I` - 打开导入对话框
  - `Ctrl/Cmd + E` - 打开导出对话框
  - `Ctrl/Cmd + U` - 打开文件上传
  - `Ctrl/Cmd + S` - 手动保存画布
  - `Ctrl/Cmd + D` - 复制选中节点
  - `Ctrl/Cmd + ,` - 打开设置
  - `Ctrl/Cmd + A` - 全选节点
  - `N` - 快速创建新节点
  - `?` - 显示快捷键帮助

- 🔔 **Toast 通知系统**
  - 自动消失的通知（默认 3 秒）
  - 支持成功、错误、警告、信息四种类型
  - 优雅的进入/退出动画
  - 可手动关闭
  - 多条通知堆叠显示

- 🎯 **节点操作增强**
  - 节点复制功能（Ctrl/Cmd + D）
  - 快速创建节点（N 键）
  - 操作反馈提示
  - 撤销/重做操作提示

- 💡 **智能 Prompt 系统**
  - 模式特定的 System Prompt

## [0.1.2] - 2024-04-17

### 修复
- 🐛 **修复节点无限变大的严重 Bug**
  - 节点创建后会不断自动变大
  - 原因：尺寸更新导致的无限循环
  - 解决：只在用户完成调整时保存尺寸
  - 添加重复更新检查
  - 设置明确的默认尺寸（200x100）

### 改进
- 🔧 优化节点尺寸管理
  - 新节点有明确的初始尺寸
  - 防止不必要的尺寸更新
  - 提升性能和稳定性

## 版本说明

### 版本号规则
- **主版本号（Major）**: 不兼容的 API 修改
- **次版本号（Minor）**: 向下兼容的功能性新增
- **修订号（Patch）**: 向下兼容的问题修正

### 变更类型
- `新增` - 新功能
- `改进` - 对现有功能的改进
- `修复` - Bug 修复
- `移除` - 移除的功能
- `废弃` - 即将移除的功能
- `安全` - 安全相关的修复
- `文档` - 文档更新
- `技术改进` - 代码重构、性能优化等
