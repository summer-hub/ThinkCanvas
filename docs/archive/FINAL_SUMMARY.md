## 🚀 如何开始测试

### 1. 启动应用

```bash
# Web 模式（推荐用于测试）
npm run dev

# Electron 模式
npm run electron:dev
```

### 2. 配置提供商

1. 点击 Header 中的 ⚙️ 按钮
2. 配置 Chat Provider（DeepSeek 或 OpenAI）
3. 配置 Embeddings Provider（Youdao 或 OpenAI）
4. 保存配置

### 3. 上传测试文档

1. 点击 "Upload" 按钮
2. 选择一个 PDF 或 Word 文档
3. 等待索引完成（查看进度条）

### 4. 测试 RAG 功能

1. 选择一个节点打开 AI Panel
2. 输入 `@` 查看自动完成
3. 选择文件并输入问题
4. 查看 AI 响应和来源归属

### 5. 运行测试套件

1. 点击 Header 中的 🎯 E2E 按钮
2. 点击 "运行所有测试"
3. 查看测试结果
