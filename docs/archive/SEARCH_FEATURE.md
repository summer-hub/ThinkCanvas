# 🔍 节点搜索功能文档

## 功能概述

Ponder AI 现在支持强大的节点搜索功能，让你可以快速找到任何想法和笔记。

---

## 🎯 功能特性

### 核心功能
- ✅ 全文搜索节点内容
- ✅ 实时高亮匹配文本
- ✅ 搜索结果计数
- ✅ 键盘导航
- ✅ 自动选中节点
- ✅ 区分节点类型（文本/AI）

### 搜索特点
- 🔤 不区分大小写
- ⚡ 实时搜索（输入即搜索）
- 🎨 高亮显示匹配文本
- 📊 显示结果数量和当前位置
- ⌨️ 完整的键盘支持

---

## ⌨️ 快捷键

### 打开搜索
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+F` (Win/Linux) | 打开搜索面板 |
| `Cmd+F` (Mac) | 打开搜索面板 |
| 点击 Header 中的 "Search" 按钮 | 打开搜索面板 |

### 搜索面板内
| 快捷键 | 功能 |
|--------|------|
| `Enter` | 下一个结果 |
| `Shift+Enter` | 上一个结果 |
| `↓` | 下一个结果 |
| `↑` | 上一个结果 |
| `Esc` | 关闭搜索面板 |

---

## 🖱️ 使用方法

### 方法 1：快捷键（推荐）
1. 按 `Ctrl/Cmd + F`
2. 输入搜索关键词
3. 按 `Enter` 浏览结果
4. 按 `Esc` 关闭

### 方法 2：点击按钮
1. 点击 Header 中的 "🔍 Search" 按钮
2. 输入搜索关键词
3. 点击 ↑↓ 按钮或使用键盘导航
4. 点击 ✕ 关闭

### 方法 3：点击结果
1. 打开搜索面板
2. 输入关键词
3. 直接点击搜索结果列表中的任意项
4. 自动跳转到该节点

## 🔧 技术实现

### 搜索算法
```typescript
// 不区分大小写的全文搜索
const searchResults = nodes.filter(node =>
  node.data.content.toLowerCase().includes(query.toLowerCase())
);
```

### 高亮实现
```typescript
// 使用正则表达式高亮匹配文本
const regex = new RegExp(`(${escapeRegex(query)})`, 'gi');
const highlighted = text.replace(
  regex,
  '<mark class="bg-canvas-accent/30">$1</mark>'
);
```

### 键盘导航
```typescript
// 循环导航
const handleNext = () => {
  setCurrentIndex((prev) => (prev + 1) % searchResults.length);
};

const handlePrevious = () => {
  setCurrentIndex((prev) => 
    (prev - 1 + searchResults.length) % searchResults.length
  );
};
```

## 🐛 已知限制

### 1. 不支持正则表达式
**当前**：只支持简单文本搜索

**未来**：可能添加高级搜索模式

### 2. 不搜索节点属性
**当前**：只搜索节点内容（content）

**不搜索**：
- 节点 ID
- 创建时间
- 颜色
- 标签（未来功能）

### 3. 不支持模糊搜索
**当前**：必须完全匹配

**示例**：
- 搜索 "AI" 不会匹配 "人工智能"
- 搜索 "idea" 不会匹配 "ideas"
