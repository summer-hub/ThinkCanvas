## 📚 API 使用

### 在组件中使用翻译

```typescript
import { useI18n } from '@/i18n';

function MyComponent() {
  const { t, locale, setLocale } = useI18n();
  
  return (
    <div>
      <h1>{t('header.appName')}</h1>
      <p>{t('aiPanel.title')}</p>
      <button onClick={() => setLocale('zh')}>
        切换到中文
      </button>
    </div>
  );
}
```

### 翻译键结构

```typescript
// 使用点号分隔的键
t('header.appName')        // "Ponder AI"
t('aiPanel.title')         // "AI Thinking Partner" / "AI 思考伙伴"
t('common.loading')        // "Loading..." / "加载中..."
```

### 添加新翻译

1. 在 `src/i18n/locales/en.ts` 添加英文
2. 在 `src/i18n/locales/zh.ts` 添加中文
3. TypeScript 会自动检查类型

## 🎯 使用指南

### 切换语言

**方法 1: 使用 UI**
1. 点击 Header 右侧的语言按钮
2. 选择想要的语言
3. 界面立即切换

**方法 2: 自动检测**
- 首次访问时自动检测浏览器语言
- 中文浏览器 → 显示中文
- 其他语言 → 显示英文

**方法 3: 编程方式**
```javascript
// 在浏览器控制台
localStorage.setItem('locale', 'zh'); // 或 'en'
location.reload();
```

### 语言设置持久化

语言选择会自动保存到 localStorage：
- 下次打开应用时保持选择
- 跨会话保持
- 可手动清除

## 🐛 已知问题

### 限制

1. **AI 响应语言**: AI 响应的语言取决于提供商和提示，不受 UI 语言影响
2. **文档内容**: 上传的文档内容不会被翻译
3. **节点内容**: 用户创建的节点内容保持原样

### 解决方案

**AI 响应语言**:
- 在提示中明确要求语言
- 例如："请用中文回答" 或 "Please answer in English"
