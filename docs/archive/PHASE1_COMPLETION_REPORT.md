## 🔧 技术实现

### 核心功能

#### 1. AI 文档分析

**函数**: `analyzeDocument(content, fileName)`

**功能**:
- 使用 AI 分析文档结构
- 识别章节和标题
- 提取核心要点
- 生成深层洞察
- 计算章节权重

**输入**: 文档内容 + 文件名  
**输出**: `DocumentStructure` 对象

**特点**:
- 结构化 JSON 输出
- 智能章节识别
- 跨章节洞察生成

---

#### 2. 知识图谱生成

**函数**: `generateKnowledgeGraph(structure, fileId)`

**功能**:
- 根据文档结构创建节点
- 自动布局（圆形）
- 节点类型区分
- 自动连接

**输入**: `DocumentStructure` + 文件 ID  
**输出**: `{ nodes, edges }`

**布局算法**:
- 中心节点: 画布中心
- 章节节点: 圆形布局（半径 400px）
- 要点节点: 围绕章节（半径 250px）
- 洞察节点: 中心上方

---

#### 3. 上传流程改进

**5 个阶段**:

| 阶段 | 图标 | 说明 | 耗时 |
|------|------|------|------|
| uploading | 📤 | 上传文件 | 1s |
| analyzing | 🔍 | AI 分析 | 5-15s |
| generating | 🎨 | 生成图谱 | 1s |
| indexing | 🔄 | 建立索引 | 3-10s |
| complete | ✅ | 完成 | - |

**用户体验**:
- 实时进度反馈
- 清晰的状态提示
- 可选的 AI 分析
- 自动错误降级

### 数据结构

#### DocumentStructure

```typescript
interface DocumentStructure {
  title: string;              // 文档标题
  summary: string;            // 整体摘要
  sections: Section[];        // 章节列表
  statistics: {
    totalWords: number;       // 总字数
    sectionCount: number;     // 章节数
    sectionWeights: Record<string, number>; // 占比
  };
  keyPoints: string[];        // 核心要点
  insights: string[];         // 深层洞察
  keywords: string[];         // 关键词
}
```

#### Section

```typescript
interface Section {
  id: string;                 // 章节 ID
  title: string;              // 章节标题
  content: string;            // 章节内容
  summary: string;            // 章节摘要
  keyPoints: string[];        // 要点列表
  wordCount: number;          // 字数
  weight: number;             // 占比 (0-1)
  level: number;              // 层级 (1/2/3)
}
```
