## 📞 使用指南

### 快速开始

1. **上传文件**
   ```
   点击 Header 中的 "Upload" 按钮
   → 拖拽文件或点击浏览
   → 等待解析完成
   → 自动创建节点
   ```

2. **查看文件**
   ```
   点击 Header 中的 "Files" 按钮
   → 浏览已上传的文件
   → 点击文件查看预览
   → 使用操作按钮管理
   ```

3. **创建节点**
   ```
   在文件列表中选择文件
   → 点击 "Create Node" 按钮
   → 节点自动添加到画布
   → 包含文件内容预览
   ```

### 支持的文件类型

- 📃 **文本文件**：.txt
- 📋 **Markdown**：.md
- 📄 **PDF**：.pdf
- 📝 **Word**：.docx, .doc

### 文件大小限制

- 单个文件最大 10MB
- 建议上传 < 5MB 的文件以获得最佳性能
