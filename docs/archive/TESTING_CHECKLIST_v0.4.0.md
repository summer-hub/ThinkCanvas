## ✅ 功能测试

### 1. 文件上传和索引

#### 1.1 基本上传
- [ ] 点击 "Upload" 按钮
- [ ] 拖拽一个 PDF 文件
- [ ] 验证：显示上传进度
- [ ] 验证：显示索引进度条
- [ ] 验证：显示百分比（0% → 100%）
- [ ] 验证：创建新节点
- [ ] 验证：文件出现在 Files 列表

#### 1.2 多种文件格式
- [ ] 上传 PDF 文件
- [ ] 上传 Word (.docx) 文件
- [ ] 上传 Text (.txt) 文件
- [ ] 上传 Markdown (.md) 文件
- [ ] 验证：所有格式都能正确解析

#### 1.3 错误处理
- [ ] 尝试上传超大文件（>10MB）
- [ ] 验证：显示错误消息
- [ ] 尝试上传不支持的格式（.jpg）
- [ ] 验证：显示错误消息

### 2. 文件引用自动完成

#### 2.1 基本功能
- [ ] 选择一个节点打开 AI Panel
- [ ] 在输入框输入 `@`
- [ ] 验证：显示文件列表下拉框
- [ ] 验证：显示文件图标（📄 PDF, 📝 Word, 📃 Text）
- [ ] 验证：最多显示 10 个文件

#### 2.2 过滤功能
- [ ] 输入 `@test`
- [ ] 验证：只显示包含 "test" 的文件
- [ ] 继续输入字符
- [ ] 验证：实时过滤更新

#### 2.3 键盘导航
- [ ] 输入 `@`
- [ ] 按 ↓ 方向键
- [ ] 验证：选中下一个文件（高亮显示）
- [ ] 按 ↑ 方向键
- [ ] 验证：选中上一个文件
- [ ] 按 Enter
- [ ] 验证：插入选中的文件名
- [ ] 验证：光标位置正确

#### 2.4 鼠标操作
- [ ] 输入 `@`
- [ ] 点击一个文件
- [ ] 验证：插入文件名
- [ ] 验证：自动完成关闭

#### 2.5 取消操作
- [ ] 输入 `@`
- [ ] 按 Escape
- [ ] 验证：自动完成关闭
- [ ] 输入 `@test`
- [ ] 输入空格
- [ ] 验证：自动完成关闭

### 3. RAG 上下文检索

#### 3.1 单文件引用
- [ ] 上传一个测试文档（例如：research.pdf）
- [ ] 等待索引完成
- [ ] 输入：`@research.pdf 这个文档的主要内容是什么？`
- [ ] 发送消息
- [ ] 验证：显示 "正在解析文件引用..."
- [ ] 验证：显示 "正在检索上下文 (1 个文件)..."
- [ ] 验证：显示 "正在增强提示..."
- [ ] 验证：显示 "正在生成响应..."
- [ ] 验证：AI 响应基于文档内容
- [ ] 验证：响应下方显示来源归属

#### 3.2 多文件引用
- [ ] 上传两个不同的文档
- [ ] 输入：`@doc1.pdf @doc2.pdf 比较这两个文档`
- [ ] 发送消息
- [ ] 验证：显示 "正在检索上下文 (2 个文件)..."
- [ ] 验证：AI 响应涉及两个文档
- [ ] 验证：来源归属显示多个文件

#### 3.3 无效文件引用
- [ ] 输入：`@nonexistent.pdf 这是什么？`
- [ ] 发送消息
- [ ] 验证：显示警告 "⚠️ 以下文件未找到或未索引: nonexistent.pdf"
- [ ] 验证：对话仍然继续（降级为无上下文模式）

### 4. 来源归属显示

#### 4.1 基本显示
- [ ] 发送带文件引用的消息
- [ ] 等待 AI 响应
- [ ] 验证：响应下方显示 "📚 来源 (N)"
- [ ] 验证：显示检索时间（例如：856ms）
- [ ] 验证：显示来源列表

#### 4.2 相似度显示
- [ ] 检查每个来源的相似度分数
- [ ] 验证：显示百分比（例如：78%）
- [ ] 验证：颜色编码正确：
  - 🟢 绿色：>80%
  - 🟡 黄色：60-80%
  - ⚪ 灰色：<60%

#### 4.3 展开/折叠
- [ ] 点击一个来源
- [ ] 验证：展开显示完整内容
- [ ] 验证：显示块索引（例如：块 #2）
- [ ] 再次点击
- [ ] 验证：折叠隐藏内容

#### 4.4 警告提示
- [ ] 发送一个与文档不太相关的问题
- [ ] 验证：如果所有来源 <60%，显示警告
- [ ] 验证：警告文本："⚠️ 所有来源的相似度较低 (<0.6)，回答可能不够准确"

#### 4.5 截断警告
- [ ] 上传一个很大的文档
- [ ] 发送需要大量上下文的问题
- [ ] 验证：如果上下文被截断，显示 "已截断" 标记

### 5. 错误处理

#### 5.1 API Key 未配置
- [ ] 清除 Embeddings API Key
- [ ] 尝试发送带文件引用的消息
- [ ] 验证：显示错误 "❌ 未配置 Embeddings API Key"

#### 5.2 检索超时

- [ ] 如果超过 2 秒
- [ ] 验证：显示 "❌ 上下文检索失败: 检索超时（超过 2 秒）"
- [ ] 验证：对话继续（降级）

#### 5.3 API 错误
- [ ] 配置一个无效的 API Key
- [ ] 尝试上传和索引文件
- [ ] 验证：显示错误消息
- [ ] 验证：文件上传成功（索引失败不影响上传）
