## 技术架构

### 数据流

```
用户输入消息
    ↓
解析 @文件引用
    ↓
检索相关上下文（2秒超时）
    ↓
增强提示
    ↓
调用 AI API
    ↓
显示响应 + 来源归属
```

### 核心模块

1. **rag.ts** - RAG 核心功能
   - 文件索引
   - 语义搜索
   - 文件引用解析

2. **ragIntegration.ts** - RAG 集成编排
   - 上下文检索编排
   - 超时处理
   - 错误分类

3. **promptEnhancement.ts** - 提示增强
   - 上下文格式化
   - 提示构建
   - Token 估算

4. **embeddingsProvider.ts** - Embeddings 提供商
   - 多提供商支持
   - JWT 认证（智谱 AI）
   - 缓存管理
