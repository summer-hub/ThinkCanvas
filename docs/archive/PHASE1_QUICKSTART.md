4. 🔄 **索引中** (3-10 秒)
5. ✅ **完成** (自动关闭)

## 🐛 遇到问题？

### 问题 1: AI 分析失败

**现象**: 显示"AI 分析失败，使用快速分析"

**原因**: 
- API Key 未配置或无效
- 网络连接问题
- AI 服务暂时不可用

**解决**:
1. 检查 API Key 是否正确
2. 检查网络连接
3. 查看浏览器控制台的错误信息

### 问题 2: 没有生成节点

**现象**: 上传完成但画布上没有节点

**原因**: 
- 文件内容为空
- 解析失败

**解决**:
1. 确保文件有内容
2. 查看控制台错误信息
3. 尝试其他文件

### 问题 3: 节点重叠

**现象**: 多个节点堆叠在一起

**原因**: 
- 节点数量过多
- 布局算法需要优化

**解决**:
1. 手动拖拽节点调整位置
2. 使用缩放功能查看全局
3. 等待 Phase 5 的布局优化
