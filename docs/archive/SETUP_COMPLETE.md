## 🎮 快速上手

### 1. 创建第一个节点
- 双击画布空白处
- 输入你的想法，比如 "如何提高工作效率"
- 按 Enter 确认

### 2. 与 AI 对话
- 点击刚创建的节点
- 右侧会打开 AI 对话面板
- 输入问题，比如 "帮我分析这个想法"
- AI 会自动回复并创建新节点

### 3. 连接想法
- 从一个节点的底部圆点拖动到另一个节点的顶部
- 建立想法之间的关联

### 4. 自定义节点
- 右键点击节点
- 选择 "Change Color" 改变颜色
- 选中节点后拖动角落调整大小

## 🐛 遇到问题？

### API 不工作
1. 检查 API Key 是否正确
2. 检查网络连接
3. 打开浏览器控制台查看错误信息
4. 尝试在设置中重新配置

### 服务器无法访问
1. 确认服务器正在运行（查看终端输出）
2. 尝试访问 http://localhost:5174/
3. 检查防火墙设置

### 数据丢失
- 数据存储在浏览器 IndexedDB 中
- 清除浏览器数据会导致丢失
- 建议定期导出重要内容（功能开发中）
