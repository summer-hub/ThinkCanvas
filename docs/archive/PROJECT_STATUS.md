## 🎯 项目概况

**Ponder AI** 是一个结合无限画布和 AI 助手的桌面思维工具，帮助用户以可视化、非线性的方式组织和探索想法。

- **当前版本**：v0.1.1
- **开发阶段**：MVP 完善阶段
- **整体进度**：35%
- **代码健康度**：✅ 优秀（无 TypeScript 错误）

## 💡 技术决策

### 已决策
1. ✅ 使用 Electron 而非 Tauri（开发速度优先）
2. ✅ 使用 Zustand 而非 Redux（简洁性）
3. ✅ 使用 IndexedDB 而非文件系统（跨平台）
4. ✅ 使用 @xyflow/react 而非自建画布（成熟度）
5. ✅ 深色主题优先（用户偏好）

### 待决策
1. ⏳ 是否添加后端服务（RAG 功能需要）
2. ⏳ 向量数据库选择（Chroma vs PGVector vs Pinecone）
3. ⏳ 是否支持插件系统
4. ⏳ 是否支持云同步
