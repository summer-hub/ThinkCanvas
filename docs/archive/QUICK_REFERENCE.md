# 🚀 v0.4.0 快速参考

## 📋 新功能速览

### 1. 文档引用 (@filename)
```
@research.pdf 这篇论文的主要结论是什么？
```
- 输入 `@` 显示文件列表
- 方向键选择，Enter 确认
- 支持多文件：`@doc1.pdf @doc2.pdf`

### 2. 来源归属
- 🟢 绿色：>80% 相似度（高质量）
- 🟡 黄色：60-80% 相似度（中等）
- ⚪ 灰色：<60% 相似度（低质量）
- 点击展开查看完整内容

### 3. 测试按钮
- 🧪 Test：基础 RAG 测试
- 🎯 E2E：端到端测试套件

---

## ⚙️ 配置指南

### Chat Provider
1. 点击 Header 的 ⚙️ 按钮
2. 选择 Chat Provider
3. 输入 API Key
4. 推荐：DeepSeek（便宜）

### Embeddings Provider
1. 点击 Header 的 ⚙️ 按钮
2. 选择 Embeddings Provider
3. 输入 API Key
4. 推荐：Youdao BCEmbedding（RAG 优化）

---

## 🎯 快速测试流程

### 5 分钟测试

1. **上传文档** (1 分钟)
   - 点击 "Upload"
   - 选择 PDF/Word 文档
   - 等待索引完成

2. **测试自动完成** (1 分钟)
   - 打开 AI Panel
   - 输入 `@`
   - 验证显示文件列表

3. **测试 RAG** (2 分钟)
   - 输入：`@filename 这个文档讲了什么？`
   - 验证 AI 响应基于文档
   - 查看来源归属

4. **运行测试** (1 分钟)
   - 点击 🎯 E2E 按钮
   - 运行所有测试
   - 验证全部通过

## 🐛 常见问题

### Q: 自动完成不显示？
A: 确保已上传并索引文件

### Q: 检索超时？
A: 正常，系统会降级为无上下文模式

### Q: 相似度都很低？
A: 重新表述问题，使用文档中的关键词

### Q: API Key 错误？
A: 检查设置中的 Embeddings Provider 配置
