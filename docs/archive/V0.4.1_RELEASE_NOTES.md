## 🚀 新功能

### 1. 对话历史管理 ✅

智能管理 AI 对话历史，提升多轮对话体验：

**功能**:
- ✅ 自动限制历史长度（最多 10 条消息）
- ✅ Token 限制（最多 4000 tokens）
- ✅ 保留系统消息（警告、错误）
- ✅ 实时显示对话统计
- ✅ 上下文使用率显示

**显示内容**:
```
AI Thinking Partner
8 条消息 • 1234 tokens (31%) • 📚 RAG
```

**技术实现**:
- 新增 `src/lib/conversationHistory.ts`
- 智能 token 估算（中文/英文分别计算）
- LRU 策略（保留最近的消息）
- 保留 RAG 上下文完整性

### 2. 查询 Embedding 缓存 ✅

LRU 缓存系统，显著提升检索速度：

**功能**:
- ✅ 最多缓存 100 个查询
- ✅ 7 天过期时间
- ✅ 持久化到 localStorage
- ✅ 按提供商和模型分类
- ✅ 缓存统计（命中率、大小）
- ✅ API Key 变更时自动清除

**性能提升**:
- 缓存命中时：**0ms** (vs 500-1000ms)
- 减少 API 调用：节省成本
- 提升用户体验：即时响应

**技术实现**:
- 新增 `src/lib/embeddingCache.ts`
- 更新 `src/lib/embeddings.ts` 使用缓存
- LRU 淘汰策略
- 自动清理过期条目

## 🎯 使用指南

### 查看对话统计

对话统计会自动显示在 AI Panel 顶部：

```
AI Thinking Partner
8 条消息 • 1234 tokens (31%) • 📚 RAG
```

**说明**:
- **8 条消息**: 当前对话中的消息数
- **1234 tokens**: 估算的 token 使用量
- **(31%)**: 占 4000 token 限制的百分比
- **📚 RAG**: 表示使用了 RAG 上下文

### 缓存管理

缓存会自动工作，无需手动管理。

**自动清除时机**:
- API Key 变更
- 提供商切换
- 7 天未使用

**手动清除** (如需要):
```javascript
import { clearEmbeddingCache } from '@/lib/embeddingCache';
clearEmbeddingCache();
```

### 查看缓存统计

在浏览器控制台：

```javascript
import { getCacheStats } from '@/lib/embeddings';
console.log(getCacheStats());
// { size: 45, maxSize: 100, hits: 120, misses: 45, hitRate: 72.7 }
```

## 🐛 已知限制

### 对话历史

1. **最大消息数**: 10 条
2. **最大 tokens**: 4000
3. **系统消息**: 最多保留 3 条

**影响**: 超长对话会自动截断，保留最近的内容

### 缓存

1. **最大缓存数**: 100 个查询
2. **过期时间**: 7 天
3. **存储限制**: 受 localStorage 限制（通常 5-10MB）

**影响**: 缓存满时会自动淘汰最旧的条目
