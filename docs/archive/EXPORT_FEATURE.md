# 📤 导出功能文档

## 功能概述

Ponder AI 现在支持将你的思维导图导出为多种格式，方便分享、备份和在其他工具中使用。

---

## 🎯 支持的格式

### 1. Markdown (.md)
- 📝 格式化的文本文档
- 📋 包含标题和结构
- 🔗 显示节点连接关系
- 📅 包含创建时间
- 💭 区分文本节点和 AI 响应

**适用场景**：
- 分享想法给他人
- 发布到博客或文档
- 在 Notion、Obsidian 等工具中查看
- 打印或转换为 PDF

### 2. JSON (.json)
- 📊 结构化数据格式
- 💾 完整的备份数据
- 🔄 可重新导入（未来功能）
- 🛠️ 可在其他工具中处理

**适用场景**：
- 数据备份
- 迁移到其他工具
- 程序化处理数据
- 版本控制

### 3. PNG (.png)
- 🖼️ 高质量图片
- 📸 当前画布截图
- 🎨 保留所有视觉效果
- 📱 易于分享

**适用场景**：
- 演示文稿
- 社交媒体分享
- 文档插图
- 快速预览

---

## 🚀 如何使用

### 方法 1：点击导出按钮
1. 点击 Header 中的 "📤 Export" 按钮
2. 选择导出格式
3. 文件自动下载

### 方法 2：快捷操作
1. 确保有节点内容
2. 点击导出
3. 选择格式
4. 完成！

## 📋 导出格式详解

### Markdown 格式

#### 文件结构
```markdown
# Ponder AI - Canvas Export

*Exported on 2024-04-17 12:30:45*

**Statistics**: 5 nodes, 3 connections

---

## 💭 Your Ideas

### 1. 第一个想法
想法的完整内容...

*Created: Apr 17, 2024, 12:00 PM*

**AI Responses:**
> AI 的回复内容...

---

### 2. 第二个想法
...

## 🔗 Connections
- "第一个想法..." → "AI 回复..."
- "第二个想法..." → "第三个想法..."
```

#### 特点
- ✅ 清晰的层次结构
- ✅ 包含时间戳
- ✅ 显示 AI 响应
- ✅ 列出所有连接
- ✅ 易于阅读和编辑

---

### JSON 格式

#### 文件结构
```json
{
  "version": "1.0",
  "exportedAt": "2024-04-17T12:30:45.000Z",
  "statistics": {
    "totalNodes": 5,
    "textNodes": 3,
    "aiNodes": 2,
    "connections": 3
  },
  "nodes": [
    {
      "id": "node_123",
      "type": "text",
      "position": { "x": 100, "y": 200 },
      "content": "节点内容",
      "createdAt": "2024-04-17T12:00:00.000Z",
      "color": "#25253a",
      "width": 200,
      "height": 100
    }
  ],
  "edges": [
    {
      "id": "edge_456",
      "source": "node_123",
      "target": "node_789"
    }
  ]
}
```

#### 特点
- ✅ 完整的数据结构
- ✅ 包含所有属性
- ✅ 易于程序处理
- ✅ 可用于备份恢复

---

### PNG 格式

#### 特点
- ✅ 2x 分辨率（高清）
- ✅ 深色背景
- ✅ 保留所有视觉效果
- ✅ 包含节点、连接、颜色

#### 技术细节
- 使用 html2canvas 库
- 背景色：#1a1a2e
- 缩放：2x（更清晰）
- 格式：PNG（无损）

### 自动命名规则
```

```

### 命名格式
- 前缀：`ponder-canvas`
- 时间戳：ISO 格式（精确到秒）
- 扩展名：根据格式

### 优点
- ✅ 不会覆盖旧文件
- ✅ 按时间排序
- ✅ 易于识别
- ✅ 支持多次导出


```typescript

export function exportToMarkdown(nodes, edges) {
  let markdown = '# Ponder AI - Canvas Export\n\n';

  const data = {
    version: '1.0',

  return JSON.stringify(data, null, 2);
}
```

## 🐛 已知限制

### 1. PNG 导出限制
- ⚠️ 只导出当前可见区域
- ⚠️ 不包含画布外的节点
- ⚠️ 需要浏览器支持 Canvas API

**解决方案**：
- 调整缩放级别以包含所有节点
- 或使用 JSON 导出完整数据

### 2. Markdown 格式限制
- ⚠️ 不保留节点位置
- ⚠️ 不保留颜色信息
- ⚠️ 连接关系简化显示

**解决方案**：
- 使用 JSON 导出保留完整信息
- Markdown 主要用于阅读和分享

### 3. 浏览器兼容性
- ⚠️ 某些旧浏览器可能不支持
- ⚠️ 需要启用 JavaScript

**支持的浏览器**：
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

## 💡 使用技巧

### 1. 定期备份
```
建议：每周导出一次 JSON 文件
原因：防止数据丢失
方法：设置提醒，定期导出
```

### 2. 多格式导出
```
建议：同时导出 Markdown 和 JSON
原因：Markdown 易读，JSON 完整
方法：连续点击两次导出
```

### 3. 导出前整理
```
建议：导出前调整画布布局
原因：PNG 导出更美观
方法：缩放和移动节点
```

### 4. 命名规范
```
建议：导出后重命名文件
原因：更容易识别内容
方法：添加项目名称前缀
```
