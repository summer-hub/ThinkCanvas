## 🎯 如何使用

### 左侧资源面板

1. **查看文件**: 展开面板查看所有文件
2. **搜索**: 输入关键词实时过滤
3. **分类**: 点击分类按钮过滤文件类型
4. **删除**: 悬停文件卡片，点击删除按钮
5. **折叠**: 点击顶部折叠按钮

### 右侧分析面板

1. **查看统计**: 点击"📈 统计"标签
2. **查看洞察**: 点击"💡 洞察"标签
3. **刷新**: 点击"刷新分析"按钮
4. **折叠**: 点击顶部折叠按钮

---

## 🐛 已知限制

1. **分析依赖节点颜色**
   - 需要使用 Phase 1 生成的节点
   - 手动节点需要设置正确的颜色

2. **文件列表不自动刷新**
   - 上传新文件后需要手动刷新
   - 点击"刷新列表"按钮

3. **分析不自动更新**
   - 修改节点后需要手动刷新
   - 点击"刷新分析"按钮
