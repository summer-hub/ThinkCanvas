# 🧪 RAG 功能测试指南

## 测试环境

- **浏览器**：Chrome/Firefox/Safari
- **URL**：http://localhost:5174/
- **版本**：v0.4.0 Phase 1

---

## 🚀 快速开始

### 1. 打开测试面板

1. 访问 http://localhost:5174/
2. 点击 Header 右侧的 **"🧪 Test"** 按钮（黄色边框）
3. 测试面板将打开

### 2. 准备测试环境

**必需条件**：
- ✅ OpenAI API Key 已配置
  - 点击 Header 中的 ⚙️ 按钮
  - 选择 OpenAI 提供商
  - 输入 API Key
  - 保存

**可选条件**（用于完整测试）：
- 📁 上传一些测试文件
  - 点击 "Upload" 按钮
  - 上传 PDF、Word 或文本文件
  - 等待上传和索引完成

---

## 📋 测试项目

### 测试 1: 文本分块 ✅

**测试内容**：
- 验证文本分块算法
- 检查块大小和重叠
- 统计分块结果

**预期结果**：
```
✅ 生成了 X 个文本块
   平均大小: ~500 字符
   总词数: XXX
```

**通过标准**：
- 生成的块数量合理
- 平均大小接近 500 字符
- 无错误信息

---

### 测试 2: API Key 检查 🔑

**测试内容**：
- 检查 OpenAI API Key 是否配置

**预期结果**：
```
✅ API Key 已配置
```

**如果失败**：
```
❌ 未找到 OpenAI API Key
   请在设置中配置 API Key
```

**解决方法**：
1. 点击 Header 中的 ⚙️ 按钮
2. 选择 OpenAI
3. 输入 API Key
4. 重新运行测试

---

### 测试 3: 嵌入生成 🧮

**测试内容**：
- 调用 OpenAI Embeddings API
- 生成文本的向量表示
- 验证嵌入维度

**预期结果**：
```
   生成嵌入向量...
✅ 嵌入维度: 1536
```

**通过标准**：
- 嵌入维度 = 1536
- 无 API 错误
- 响应时间 < 5 秒

**可能的错误**：
- `❌ 嵌入生成失败: Invalid API key`
  - 解决：检查 API Key 是否正确
- `❌ 嵌入生成失败: Rate limit exceeded`
  - 解决：等待几分钟后重试
- `❌ 嵌入生成失败: Network error`
  - 解决：检查网络连接

---

### 测试 4: 相似度计算 📊

**测试内容**：
- 计算语义相似的文本对
- 计算语义不相似的文本对
- 验证相似度分数

**测试用例**：
1. **高相关文本**：
   - "机器学习是人工智能的一个分支"
   - "深度学习是机器学习的一种方法"
   - 预期相似度：> 0.7

2. **低相关文本**：
   - "机器学习是人工智能的一个分支"
   - "今天天气很好"
   - 预期相似度：< 0.5

**预期结果**：
```
   "机器学习..." vs "深度学习..."
   相似度: 0.8234 ✅ 高相关
   "机器学习..." vs "今天天气..."
   相似度: 0.1234 ✅ 低相关
```

**通过标准**：
- 相关文本相似度 > 0.7
- 不相关文本相似度 < 0.5
- 相似度在 [0, 1] 范围内

---

### 测试 6: 语义搜索 🔍

**测试内容**：

- 返回 Top-K 结果

**预期结果（有索引文件）**：
```

   1. example.pdf

   2. notes.txt
      相似度: 0.7456

      相似度: 0.6789
      内容: 机器学习算法的应用...
```

**预期结果（无索引文件）**：

**通过标准**：
- 返回的结果与查询相关
- 相似度按降序排列
- 相似度 > 0.5（默认阈值）

## 🐛 常见问题

### Q1: "❌ 未找到 OpenAI API Key"

**原因**：未配置 API Key

**解决**：
1. 点击 Header 中的 ⚙️
2. 选择 OpenAI
3. 输入有效的 API Key
4. 点击保存
5. 重新运行测试

---

### Q2: "❌ 嵌入生成失败: Invalid API key"

**原因**：API Key 无效或过期

**解决**：
1. 访问 https://platform.openai.com/api-keys
2. 检查 API Key 是否有效
3. 生成新的 API Key
4. 更新配置
5. 重新运行测试

---

### Q3: "⚠️ 没有找到相关结果"

**原因**：没有索引的文件或查询不匹配

**解决**：
1. 上传一些测试文件
2. 等待索引完成
3. 使用与文件内容相关的查询
4. 重新运行测试

---

### Q4: 相似度计算结果异常

**原因**：可能是 API 响应问题或文本质量问题

**解决**：
1. 检查网络连接
2. 重新运行测试
3. 尝试不同的测试文本
4. 检查 API 配额

---

### Q5: 索引进度卡住

**原因**：API 限流或网络问题

**解决**：
1. 等待几分钟
2. 检查网络连接
3. 检查 API 配额
4. 重新运行测试
