# ⌨️ 键盘快捷键 | Keyboard Shortcuts

ThinkCanvas 提供丰富的键盘快捷键，帮助你更高效地工作。

---

## 📝 编辑 | Editing

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `N` | 创建新节点 | 在画布中心快速创建节点 |
| `Delete` / `Backspace` | 删除节点 | 删除当前选中的节点 |
| `Ctrl/Cmd + D` | 复制节点 | 复制选中节点（自动偏移位置） |
| `Ctrl/Cmd + Z` | 撤销 | 撤销上一步操作 |
| `Ctrl/Cmd + Shift + Z` | 重做 | 重做已撤销的操作 |
| `Ctrl/Cmd + Y` | 重做 | 重做已撤销的操作（备选） |
| `Escape` | 取消选择 | 取消当前节点选择 |

---

## 🔍 导航 | Navigation

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl/Cmd + F` | 搜索 | 打开节点搜索面板 |
| `Ctrl/Cmd + A` | 全选 | 选择所有节点 |
| `Enter` | 下一个搜索结果 | 在搜索面板中跳转到下一个 |
| `Shift + Enter` | 上一个搜索结果 | 在搜索面板中跳转到上一个 |
| `↑` / `↓` | 导航搜索结果 | 使用方向键浏览搜索结果 |

---

## 📁 文件 | File

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl/Cmd + S` | 保存 | 手动保存画布到本地 |
| `Ctrl/Cmd + I` | 导入 | 打开文本导入对话框 |
| `Ctrl/Cmd + E` | 导出 | 打开导出对话框（MD/JSON/PNG） |
| `Ctrl/Cmd + U` | 上传文件 | 打开文件上传对话框 |

---

## ⚙️ 其他 | Other

| 快捷键 | 功能 | 说明 |
|--------|------|------|

| `?` | 显示快捷键 | 显示所有可用快捷键 |

|------|------|
| 双击画布 | 创建新节点 |

## 💡 使用技巧 | Tips

### 1. 快速创建节点
- 按 `N` 键在画布中心创建节点
- 双击画布空白处创建节点
- 使用 `Ctrl/Cmd + I` 批量导入节点

### 2. 高效编辑
- 使用 `Ctrl/Cmd + D` 快速复制节点
- 使用 `Ctrl/Cmd + Z/Y` 撤销/重做
- 使用 `Delete` 快速删除节点

### 3. 快速导航
- 使用 `Ctrl/Cmd + F` 搜索节点
- 使用 `Enter` / `Shift + Enter` 浏览搜索结果
- 使用 `Escape` 关闭面板

### 4. 文件管理
- 使用 `Ctrl/Cmd + S` 手动保存（自动保存已启用）
- 使用 `Ctrl/Cmd + E` 导出为 Markdown/JSON/PNG
- 使用 `Ctrl/Cmd + U` 上传 PDF/Word 文档

### 5. AI 对话
- 选中节点后自动打开 AI 面板
- 使用 `@文件名` 引用已上传的文档
- 切换 Agent/Inspire 模式获得不同风格的回答

## 🎯 快捷键冲突处理

### 输入框中的快捷键
当你在输入框（textarea、input）中输入时，大部分快捷键会被禁用，避免误触发。

**例外**:
- `Ctrl/Cmd + F` - 搜索（始终可用）
- `Ctrl/Cmd + ,` - 设置（始终可用）

### 浏览器快捷键
某些快捷键可能与浏览器冲突：
- `Ctrl/Cmd + S` - 浏览器保存页面（已被覆盖）
- `Ctrl/Cmd + F` - 浏览器查找（已被覆盖）

## 📱 平台差异 | Platform Differences

### macOS
- 使用 `Cmd` 键（⌘）
- 使用 `Delete` 键删除

### Windows/Linux
- 使用 `Ctrl` 键
- 使用 `Delete` 或 `Backspace` 删除
